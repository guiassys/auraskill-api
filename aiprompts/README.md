# AI Prompts

This directory contains all the resources related to the AI-powered development process, including guidelines, personas, and tasks.

## Directory Structure

-   `guidelines/`: Contains the core principles and technical standards that the AI agent must follow.
-   `personas/`: Defines the specific roles and expertise the AI agent should adopt.
-   `tasks/`: Holds the individual task prompts that are assigned to the AI agent.

## How to Create a New Task

To generate a new task, follow these steps:

1.  **Directory:** Create a new Markdown file in the `@/aiprompts/tasks/` directory.
2.  **Filename:** The filename must be prefixed with a sequential number (e.g., `004-new-task-title.md`).
3.  **Template:** Use the following standard template for the task content:
    *   **Title:**
    *   **Objective:**
    *   **Expected Result:**
